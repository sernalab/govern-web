# Impeccable Commands — Skill de Refinamiento

Sistema de 17 comandos para mejorar la calidad de diseño, UX y codigo de forma estructurada.
Basado en [impeccable.style](https://impeccable.style/cheatsheet).

## Cuando usar esta skill

Cualquier agente puede aplicar estos comandos cuando:
- Se necesita evaluar calidad antes de shipping (`audit`, `critique`)
- Se quiere refinar un componente o feature terminada (`polish`, `harden`, `optimize`)
- Se necesita ajustar intensidad visual o tonal (`quieter`, `bolder`)
- Se quiere mejorar copy, legibilidad o adaptacion (`clarify`, `distill`, `adapt`)
- Se busca anadir personalidad o movimiento (`animate`, `delight`, `colorize`)

## Flujo recomendado

```
audit/critique → normalize/polish/harden → optimize → ship
```

1. **Diagnosticar** — Identifica problemas con `audit` (tecnico) o `critique` (UX/diseno)
2. **Corregir** — Aplica `normalize`, `polish`, `harden` segun hallazgos
3. **Optimizar** — Performance y bundle con `optimize`
4. **Ajustar tono** — `quieter`/`bolder` si el resultado visual lo necesita
