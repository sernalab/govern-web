# Agent Teams — Govern Web

> Habla con el CEO. Él decide a quién delegar.

## Cómo usar
Abre Claude Code en este proyecto y di:
```
Usa el agente CEO para [tu petición]
```

## Equipo disponible (10 agentes)

### Liderazgo
1. **CEO** — Orquestador estratégico, delega al equipo, prioriza

### Engineering & Product
2. **CTO** — Arquitectura, tech decisions, code review
3. **Frontend Dev** — Implementación Astro/TypeScript/Tailwind
4. **Data Engineer** — Integración APIs públicas, scrapers, pipelines de datos
5. **QA** — Testing, seguridad, accesibilidad, edge cases
6. **Product Manager** — PRDs, user stories, roadmap

### Design & UX
7. **UX Designer** — Diseño visual, dashboards, data visualization
8. **UX Auditor** — Auditorías de usabilidad, heurísticas de Nielsen

### Growth & Legal
9. **Growth** — SEO, marketing, audiencia, launch plan
10. **Legal Advisor** — RGPD, derecho al honor, datos judiciales, disclaimers

## Outputs
Los agentes guardan sus informes en:
- `docs/audits/` — Auditorías (UX, seguridad, SEO, legal, code review)
- `docs/prd/` — Product Requirements Documents
- `docs/data/` — Documentación de fuentes de datos y APIs
- `docs/legal/` — Documentos legales, disclaimers, análisis RGPD
- `docs/growth/` — Estrategia de lanzamiento, SEO
- `docs/roadmap.md` — Roadmap del producto
